# 🚀 Indian Startup Funding Analysis using Pandas

This project demonstrates data cleaning, transformation, filtering, aggregation, and exploratory data analysis using the Pandas library.

## 📌 Objective

Analyze an Indian startup funding dataset by performing real-world data cleaning and answering business-oriented analytical questions.

## 🛠️ Technologies Used

- Python
- Pandas
- Google Colab
- Jupyter Notebook

## 📊 Skills Demonstrated

- Data Cleaning
- Missing Value Handling
- Data Type Conversion
- String Operations
- Filtering
- Sorting
- GroupBy & Aggregation
- Crosstab Analysis
- Derived Columns
- Business Insights

## 📈 Dataset Features

- 468 startup funding records
- Messy city names
- Industry variants
- Funding stage variants
- Mixed date formats
- Missing values
- Duplicate records

## 📚 Pandas Concepts Used

- rename()
- replace()
- drop_duplicates()
- fillna()
- dropna()
- to_numeric()
- to_datetime()
- apply()
- lambda
- groupby()
- agg()
- idxmax()
- crosstab()
- cut()
- isin()
- between()
- sort_values()
- value_counts()

## 📌 Key Business Questions Solved

- Which city attracts the highest funding?
- Which industry has the highest average funding?
- Market saturation analysis
- Emerging startup identification
- Investor analysis
- Funding efficiency calculation
